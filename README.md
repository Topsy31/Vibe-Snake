# Photo Snake Game

A classic Snake game with a twist - eat your own photos instead of regular food!

## Play the Game

### Original Version (Before)
**[Play Original Version](https://topsy31.github.io/Vibe-Snake/)**
- Basic photo snake game
- Simple red circle food (or photos)
- Keyboard controls only

### Enhanced Version (After)
**[Play Enhanced Version](https://topsy31.github.io/Vibe-Snake/index2.html)**
- Dynamic background changes to eaten photos
- Pulsing dayglow green halo around food
- Swipe controls for mobile devices
- Keyboard controls (arrow keys)

## Features

- Upload your own photos to use as food items
- Classic snake gameplay
- Score tracking with localStorage
- Responsive design

## How to Play

1. Upload photos using the file picker
2. Use arrow keys (or swipe on mobile for enhanced version) to control the snake
3. Eat the photos to grow and score points
4. Don't hit the walls or yourself!
